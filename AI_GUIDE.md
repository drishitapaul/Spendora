# AI Guidance Notes

## Purpose
AI tools were used to assist in development, debugging, and structuring the application.

## Constraints Applied
- Keep code simple and readable
- Avoid over-engineering
- Ensure all API endpoints are testable
- Validate all inputs before database operations

## Verification Process
- All AI-generated code was manually reviewed
- API endpoints tested using browser and frontend
- Ensured correct data flow between frontend and backend

## Risk Awareness
- Avoid blindly trusting AI-generated code
- Ensure logic correctness before integration
